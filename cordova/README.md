# whu-library-seat-mobile/cordova

> 本文件夹包含一个 Cordova 工程，用于将 www 文件夹内的网页打包为移动端的安装包

## 构建命令

``` bash
# 添加 Android 平台
cordova platform add android

# 拷贝资源文件
cordova prepare android

# 打包应用程序(debug 版本)
cordova build android
```
